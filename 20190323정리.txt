좋은 자료구조 : 알고리즘을 덜사용하는 자료구조
좋은 알고리즘 : 저장공간을 적게 잡아먹는 알고리즘
기계어 -> 어셈블리어 -> C/C++ -> 자바,C#,Python,Ruby

기계가 알기 쉬운 코드(메모리, CPU)  --------  중간수준(C/C++) -------- 사람이 알기 쉬운 코드

자료구조 : 1개 이상의 자료를 다루는 것

배열/링크드리스트/스택/큐

배열: 어디에 접근하든 항상 같은 속도로 접근할 수 있다.
링크드리스트: 임의접근해야함.
스택: 후입선출(ex. 접시  -> 설거지하고 쌓아놓은 접시를 맨 위의 것부터 사용함)
큐 : 선입선출(ex. 수도관 -> 가장 먼저 들어온 물이 가장 먼저 나감)
 

Component 

Rigidbody - 물리(무게/질량)계산
Capsule Colider - 캡슐형 충돌체
		isTrigger = 충돌시 트리거 발동
